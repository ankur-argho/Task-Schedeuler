    Skill Development Project

    A modern Task Scheduler web application built with a responsive HTML, CSS and JavaScript frontend and a powerful C++ backend implementing Data Structures and Algorithms.

    Project Overview

    This project helps users organize daily tasks based on priority. The frontend provides a smooth and responsive user experience, while all business logic, task management, searching, sorting, scheduling, and file handling are implemented in C++ using Data Structures and Algorithms.
